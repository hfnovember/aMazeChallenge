Το avatar σας θα μετακινήται σύμφωνα με τις προγραμματισμένες οδηγίες σας. Μόλις ξεκινήσει το παιχνίδι, δεν θα μπορείτε να επιρεάσετε το avatar σας. Θα εκτελέσει τις οδηγίες σας για να προσπαθήσει να βρει μια διέξοδο από το λαβύρινθο, σαν ένα ρομπότ.
